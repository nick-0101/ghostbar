---
title: How to activate GhostBar
---

To activate GhostBar, just click the following keyboard shortcut depending on your device:

| Platform | Command                                                                                   |
| -------- | ----------------------------------------------------------------------------------------- |
| MacOS    | <kbd><Icon icon="option" iconType="duotone" /> Option</kbd>+<kbd>Shift</kbd>+<kbd>I</kbd> |
| Linux    | <kbd><Icon icon="option" iconType="duotone" /> Option</kbd>+<kbd>Shift</kbd>+<kbd>I</kbd> |
| Windows  | <kbd><Icon icon="alt" iconType="duotone" /> Alt</kbd>+<kbd>Shift</kbd>+<kbd>G</kbd>       |

This will bring up a textbox at the bottom of the page as well as a selector which you can use to highlight any text to query.

<img src="/images/textarea-demo.png" alt="Textarea activated" height="300" className="rounded-lg" />

Then, just type out your query and click the submit button!
