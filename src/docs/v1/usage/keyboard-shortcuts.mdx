---
title: Keyboard Shortcuts
description: Overview of the available keyboard shortcuts
---

To activate GhostBar:
| Platform | Command |
| -------- | ------------------------------------------------------------------------ |
| MacOS | <kbd><Icon icon="option" iconType="duotone" /> Option</kbd>+<kbd>I</kbd> |
| Linux | <kbd><Icon icon="option" iconType="duotone" /> Option</kbd>+<kbd>I</kbd> |
| Windows | <kbd><Icon icon="alt" iconType="duotone" /> Alt</kbd>+<kbd>I</kbd> |

`Esc`: To close prompt popup
